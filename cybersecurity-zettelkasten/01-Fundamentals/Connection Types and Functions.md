# Connection Types and Functions

**Created:** 2025-11-25
**Tags:** #fundamentals #networking #hardware #connectivity

## Description

Different types of physical and logical connections enable devices to communicate. Understanding connection types is essential for troubleshooting, network design, and security implementation.

## Key Concepts

### Physical Connections
- **Ethernet**: Wired network connections (Cat5e, Cat6, Cat7)
- **USB**: Universal Serial Bus for peripherals
- **HDMI/DisplayPort**: Video and audio connections
- **Fiber Optic**: High-speed data transmission
- **Serial/Parallel**: Legacy connection types

### Wireless Connections
- [[Wireless Technologies]] - WiFi, Bluetooth, NFC, Infrared
- Radio frequency communication
- Wireless standards and protocols

### Network Interface Types
- Network Interface Cards (NICs)
- Wireless adapters
- Virtual network interfaces

## Security Considerations

- **Physical Access Control**: Securing physical connection points
- **Wireless Security**: Encryption and authentication for wireless
- **USB Security**: Preventing malicious USB devices
- **Network Segmentation**: Using different connection types for isolation
